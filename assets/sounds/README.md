# Learnova Sound Assets

Add these .mp3 files here before APK build:

| File         | When it plays              | Style             |
|-------------|---------------------------|-------------------|
| tap.mp3     | Any button tap             | Soft UI click     |
| xp.mp3      | XP coin animation fires    | Cheerful chime    |
| correct.mp3 | Quiz correct answer        | Bright ding       |
| wrong.mp3   | Quiz wrong answer          | Gentle low tone   |
| level-up.mp3| Level-up celebration fires | Short fanfare     |
| streak.mp3  | Streak celebration fires   | Energetic jingle  |
| unlock.mp3  | Avatar unlock toast fires  | Magical shimmer   |

Recommended free sources:
- https://freesound.org
- https://mixkit.co/free-sound-effects/
- https://pixabay.com/sound-effects/

Keep each file under 100KB.
Export as MP3, 44.1kHz, mono.
